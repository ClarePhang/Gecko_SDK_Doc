uint32_t CSEN_InitMode_TypeDef::inputMask0

Mask selects inputs 0 to 31. Effect depends on sampleMode. If sample mode is bonded, then mask selects inputs to short together. If sample mode is scan, then mask selects which inputs will be scanned. If sample mode is single and auto-ground is on (autoGnd is true), mask selects which pins are grounded.
